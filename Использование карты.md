Этот гайд объединяет всё, что нужно для работы с картой лагеря.

## Показ карты

Сбросьте старые зоны, затем объявите доступные зоны через `set_zone(name, label)`, после — покажите карту.

Пример:

```python
label map_my_mod:
    $ disable_all_zones()         # очистить доступные зоны

    $ set_zone("clubs", "day1_gone_to_clubs")

    $ show_map()                  # показать карту
    return
```

Вот список всех зон и как они обозначены в игре:

- `me_mt_house` — Домик Семёна и Ольги Дмитриевны
- `estrade` — Сцена
- `music_club` — Музклуб
- `square` — Площадь
- `dining_hall` — Столовая
- `sport_area` — Спорткомплекс
- `beach` — Пляж
- `boat_station` — Лодочный причал
- `clubs` — Клубы
- `library` — Библиотека
- `medic_house` — Медпункт
- `camp_entrance` — Ворота в лагерь
- `forest` — Лес

Пример карты, на которой будут включены все зоны:

```python
label map_my_mod:
    $ disable_all_zones()

    $ set_zone("me_mt_house",  "day1_gone_to_mt_house")  # Домик Семёна и ОД
    $ set_zone("estrade",      "day1_gone_to_estrade")   # Сцена
    $ set_zone("music_club",   "day1_gone_to_music_club") # Музклуб
    $ set_zone("square",       "day1_gone_to_square") # Площадь
    $ set_zone("dining_hall",  "day1_gone_to_dining_hall") # Столовая
    $ set_zone("sport_area",   "day1_gone_to_sport_area") # Спорт. комплекс
    $ set_zone("beach",        "day1_gone_to_beach") # Пляж
    $ set_zone("boat_station", "day1_gone_to_boat_station") # Лодочная станция
    $ set_zone("clubs",        "day1_gone_to_clubs") # Клубы
    $ set_zone("library",      "day1_gone_to_library") # Библиотека
    $ set_zone("medic_house",  "day1_gone_to_medic_house") # Медпункт
    $ set_zone("camp_entrance","day1_gone_to_camp_entrance") # Ворота
    $ set_zone("forest",       "day1_gone_to_forest") # Лес

    $ show_map()
    return
```

## Обработка клика по зоне

При клике движок сохраняет выбранную зону и делает переход на указанный вами лейбл. В лейбле зоны обычно выключают текущую зону, чтобы не зациклиться на карте:

```python
label day1_gone_to_clubs:
    $ disable_current_zone()   # зона больше не кликабельна после прохождения этого лейбла карты
    "Ты приходишь в клубы..."
    jump map_my_mod # незабудьте прыгнуть обратно на лейбл с картой
```

Если зона должна оставаться активной после возврата на карту — используйте вместо этого `reset_current_zone()` (сбрасывает на дефолтное состояние и при этом оставляет доступной).


:::warning
**ВАЖНО!**

Настоятельно рекомендуем разделить лейбл, в котором добавляются зоны на карту и показывается сама карта на две: 1) указываем зоны; 2) показываем карту. Это нужно для того, чтобы если у Вас имеется карта с множественным прохождением по локациям, после возвращения на карту у Вас вновь не включались все доступные локации для посещения.

```python
label map_my_mod_prepare: # Здесь мы выключаем предыдущие зоны и включаем свои
  $ disable_all_zones()
  $ set_zone("square", "square_label")
  $ set_zone("beach", "beach_label")
  jump map_my_mod

label map_my_mod: # Здесь мы показываем карту
  $ show_map()

label square_label:
  $ disable_current_zone() # Выключаем текущую зону  
  "Площадь!"
  jump map_my_mod # После возвращения на карту у нас выключиться посещение площади
```

:::

## Чибики на карте

Можно назначить в зоне иконку-подсказку (чибика):

```renpy
$ set_chibi("square", "sl")   # показать Славю на площади
$ reset_chibi("square")        # убрать иконку
```

Доступные чибики:

- `?` - знак вопроса
- `me` - Семён
- `mi` - Мику
- `sh` - Шурик
- `el` - Электроник
- `mz` - Женя
- `mt` - Ольга Дмитриевна
- `uv` - Юля
- `un` - Лена
- `us` - Ульяна
- `dv` - Алиса
- `sl` - Славя
- `cs`- Виола

## Функции

- `disable_all_zones()` / `enable_all_zones()` — выключить/включить все зоны.
- `set_zone(name, label)` — сделать зону активной и назначить лейбл для этой зоны.
- `reset_zone(name)` — сбросить зону (сделать неактивной и вернуть к дефолту).
- `enable_empty_zone(name)` — включить зону без назначения отдельного лейбла (использует дефолт).
- `reset_current_zone()` / `disable_current_zone()` — сбросить/выключить зону, по которой только что кликнули.
- `been_there()` — функция-проверка, показывает сколько раз игрок уже кликал в текущую зону (удобно для повторных посещений и проверки на это).
- `set_chibi(name, ch)` / `reset_chibi(name)` — назначить/сбросить чибика на зоне.
- `show_map()` — показать карту.

## Дополнительно

- Повторные визиты:

  ```python
  label day1_gone_to_square:
      if been_there() > 1:
          "Ты вновь на площади."
      else:
          "Ты впервые на площади."
      $ disable_current_zone()
      return
  ```
- Оставить зону активной, сбросить счётчик `been_there`: используйте `reset_current_zone()` вместо `disable_current_zone()`.

## Подводные камни

- Всегда включайте зоны через `set_zone()` перед показом карты; пустая карта без активных зон не позволит продвинуться.
- В лейбле зоны не забудьте убирать текущую зону (`disable_current_zone()`), чтобы предотвратить повторный клик по той же зоне без изменения сценария.
